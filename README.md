# Healthcare Clinic DBMS & Tableau Analytics Project

## Project Overview

This project was developed to support chronic disease monitoring and healthcare analytics using a relational database system and interactive Tableau dashboards.

The system was designed for healthcare clinic environments to manage patient visits, diagnoses, vital signs, medications, and follow-up appointments for chronic conditions such as diabetes, hypertension, and high cholesterol.

The project combines:
- relational database design in MySQL
- SQL querying and reporting
- healthcare analytics
- Tableau dashboard visualization

# Database Features

The MySQL database includes:

- 8 normalized relational tables
- primary and foreign key constraints
- role-based access control
- SQL views
- analytical reporting queries

Core entities include:
- Patients
- Doctors
- Departments
- Visits
- Vitals
- Diagnoses
- Medications
- FollowUps
